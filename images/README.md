# Hebrew Letter Images

The app currently uses public Google Drive thumbnail URLs from the Drive folder named `Hebrew Alphabet`.

To make the app fully local/offline, download or export each image into this folder using the filenames listed in `alphabet-data.js`, such as:

- `aleph.png`
- `bet.png`
- `gimel.png`
- `dalet.png`
- `hey.png`
- `vav.png`
- `zayin.png`
- `het.png`
- `tet.png`
- `yod.png`
- `kaf.png`
- `lamed.png`
- `mem.png`
- `nun.png`
- `samekh.png`
- `ayin.png`
- `pe.png`
- `tsadi.png`
- `qof.png`
- `resh.png`
- `shin.png`
- `tav.png`

Each card first tries its local filename, then falls back to the Google Drive image URL.
